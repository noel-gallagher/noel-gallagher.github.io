What happens when we start our computers?

Firstly, after powering up, the computer executes BIOS/UEFI - this is essentially a mini program which lives on the motherboard's ROM(UEFI is a bit different). It is responsible for performing POST checks and intialising hardware.

Next, depending on boot order, a program is loaded and executed from the master boot record; the first 512 bytes of storage.

The code in this 512 bytes is typically the bootloader, used to load the OS.

UEFI does not rely on the master boot record to load the bootloader, it can for compatibility, but rather it can load a much larger and sophisticated program stored on EFI system partition.
